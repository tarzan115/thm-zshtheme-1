# thm-zshtheme
Another yet theme for Oh-My-Zsh<br>
A modified 'The Poncho' theme

## Installation guide
```bash
wget -P ~/.oh-my-zsh/themes https://raw.githubusercontent.com/thm-unix/thm-zshtheme/master/thm.zsh-theme
```
* Set ZSH_THEME="thm" in ~/.zshrc<br>
